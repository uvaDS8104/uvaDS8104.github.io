About the course

Networks provide a unifying framework to study the structure hidden within complex data. This graduate-level course focuses on the fundamental concepts and statistics as well as recent advancements and applications of network science. Topics include: graph theory, structural paradoxes, measures and algorithms for quantifying importance, community detection, network inference, recommendation systems, and link prediction.

What You Will Learn Along the Way

In this class, you’ll learn how to think like a network scientist and become familar with the tools we use to uncover the secerts of network structures.

By the end of the course, students are expected to be able to identify, construct, and analyze networks by choosing and applying appropriate methods and algorithms. Students are also expected to be able to explain, both mathematically and conceptually, the key network concepts and statistical properties, and their implications.

As an upper-level PhD course, students will also be exposed to scientific communication and open debates in network science. We will discuss soft science skills, including argumentation, presentation, and visual storytelling.

Course Structure

This course employs a combination of instructor lead lectures and student-lead discussion. Students are required to read assigned readings, attend (watch) the lectures, complete their project assignments, and engage in (online) discussions.

Every week, each student will be required to introduce an article and moderate the class’s discussion, and all students are expected to participate. Technical evaluation will be based on a final project. Students may choose any network-related topics that involve network analysis or modeling, although it is strongly encouraged to discuss with the instructor about the project topics.

Textbooks

There are several fantastic treatments of Network Science. None of these are required readings, but they have all greatly influenced the way I think about networks.

  • Barabasi (2016) Network Science. A comprehensive and easy to read guided tour. Its Free or in print at Amazon.

  • Newman (2018) Networks: An Introduction. The network bible, now updated! Its a book you will continuely reference. Amazon

  • Easley & Kleinberg (2010) Networks, crowds, and markets: Reasoning about a highly connected world. A wonderful and easy to read overview of network science with a focus more towards social networks and organizations. The full book is here. or on Amazon.

  • Menczer, Fortunato & Davis (2020) A First Course in Network Science Amazon

Online help

Data science and statistical programming can be difficult. Computers are stupid and little errors in your code can cause hours of headache (even if you’ve been doing this stuff for years!).

Fortunately there are tons of online resources to help you with this. The most important is StackOverflow (a Q&A site with hundreds of thousands of answers to all sorts of programming questions).

  • Dive Into Python by Mark Pilgrim (available online): a good Python book.

  • An introduction to statistics (with Python) by Thomas Haslwanter (available online): this book uses Python to explain basic statistics. It also contains a succinct tutorial for Python and data visualization using Python.

  • Learn Git and Github

  • Networkx Basics
